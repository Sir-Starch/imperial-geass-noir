## Geass Wallpaper Symbol

The central Geass-like symbol used in the wallpapers is derived from Wikimedia
Commons `File:Geass.svg`.

- Source: https://commons.wikimedia.org/wiki/File:Geass.svg
- Direct SVG URL: https://upload.wikimedia.org/wikipedia/commons/6/65/Geass.svg
- Author: Koveras at English Wikipedia
- Wikimedia description: “The Geass symbol ... based on artworks of the
  television series Code Geass (2006-2007).”
- Listed licenses on Wikimedia Commons: GFDL-1.2-or-later and
  CC BY-SA 1.0/2.0/2.5/3.0, with license choice.
- License selected for this package’s derived wallpaper use: CC BY-SA 3.0.

The symbol has been recolored, darkened, scaled, and integrated into original
abstract wallpapers. Because the source is explicitly based on Code Geass
artwork, downstream users should treat the wallpaper symbol as a third-party
franchise-derived asset with its own license and possible trademark or other
rights considerations. The rest of the theme remains usable without relying on
official anime screenshots, character art, or logos.

## Icon Theme

`kde/icons/ImperialGeassNoir/` is a recolored derivative of the user-provided
Papirus Vectorel/Papirus-Adapta-Nokto icon theme archive. The source archive
included LGPL-3.0 license text and contributor metadata. The derived icon theme
keeps that licensing and attribution context.
