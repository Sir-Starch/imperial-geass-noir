# Assets

This directory is reserved for optional local additions such as screenshots or
extra wallpaper exports.

The package intentionally ships no anime screenshots, character art, or official
logos.

The application launcher icon uses the crest extracted unchanged from Wikimedia
Commons `File:Holy Britannian Empire.svg`; the flag background is not included.

- Source: https://commons.wikimedia.org/wiki/File:Holy_Britannian_Empire.svg
- Author: Jud nereide
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
- Local asset: `assets/holy-britannian-crest.svg`

The wallpapers now include a darkened central sigil derived from Wikimedia
Commons `File:Geass.svg`:

- Source: https://commons.wikimedia.org/wiki/File:Geass.svg
- Direct SVG: https://upload.wikimedia.org/wikipedia/commons/6/65/Geass.svg
- Author: Koveras at English Wikipedia
- Wikimedia-listed licenses: GFDL-1.2-or-later and CC BY-SA
  1.0/2.0/2.5/3.0, with license choice
- Selected license for this package's derivative wallpaper use: CC BY-SA 3.0

The Commons description states that the source symbol is based on artwork from
the Code Geass television series. Treat that wallpaper symbol as a separate
third-party franchise-derived asset with its own attribution and possible
trademark or other rights considerations. See `NOTICE.md`.

The Aurorae window decoration is now based on the local Daemon-2.0 Aurorae
theme by Mathis Paquet (`GPL_V3` in its metadata). Imperial Geass Noir recolors
it to the Noir purple palette, keeps Daemon's working glow/geometry, and changes
the top/titlebar treatment plus centered title text.

The icon theme in `kde/icons/ImperialGeassNoir` is a full recolored derivative
of the user-provided Papirus Vectorel/Papirus-Adapta-Nokto package. Its original
license is LGPL-3.0 as provided in
`Papirus-Icon-Theme-Vectorel-Folders-master.zip`; the Imperial Geass Noir
changes rename the package and adapt the SVG colors to the Noir palette.
