# Imperial Geass Noir Palette

| Role | Hex | Notes |
| --- | --- | --- |
| Background | `#07070B` | Desktop, terminal, darkest surfaces |
| Surface | `#111018` | Windows, menus, primary widgets |
| SurfaceAlt | `#191522` | Raised widgets and alternate rows |
| Panel | `#0D0B12` | Top command bar base, used with transparency |
| Purple | `#6046A6` | Primary accent, focused decoration |
| Crimson | `#9E102B` | Selection and destructive emphasis |
| Gold | `#C4A45A` | Premium linework, cursor, active tab |
| Text | `#E8E1D2` | Main foreground |
| MutedText | `#A49AAD` | Secondary text |
| Border | `#2B2238` | Soft widget borders |
| DangerGlow | `#D33A52` | Sparse glow/highlight only |

Design notes:

- Dark imperial rebellion, not bright neon.
- Abstract chess, crest, moon, and HUD geometry only.
- No official anime assets, logos, screenshots, character art, or copied marks.
