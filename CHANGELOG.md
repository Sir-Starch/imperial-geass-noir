# Changelog

## 1.0.0

Initial release of Imperial Geass Noir.

- KDE Plasma 6 Global Theme, Plasma Style, color scheme, icon theme, and
  original Aurorae window decoration.
- Full-size icon pack rebuilt from untouched Papirus source through a
  reproducible build/recolor pipeline, covering application, tray, status,
  device, place, mimetype, and symbolic icons instead of only panel assets.
- Top-only Plasma command-bar layout.
- Gold-default wallpapers with per-monitor standard/ultrawide selection, plus
  purple and dark-crimson alternatives.
- Kitty, Konsole, Alacritty, Ghostty, and WezTerm terminal profiles.
- zsh, bash, fish, and Starship prompt profiles.
- IBM Plex Sans / IBM Plex Mono typography defaults.
- GPL-3.0-or-later project licensing with third-party notices for derived
  Aurorae, icon, and wallpaper assets.
